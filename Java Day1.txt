1.Integer:
__________
Program
__________

Input:-

public class Datatypes{
  public static void main(String[] args){
     int age =21;
     System.out.println(age);
}
}


Output:-
________

21

int range is [-2Billion,2Billion]
________
2.byte
_________

To represent age if use byte data type we can reduce the storage in ram
because max avg age of a human is below 100 so that we don't require 2 billion data space so we use byte data type
whose range is 

byte range is [-128,127]

Program:-
_________

Input:-

public class Datatypes{
  public static void main(String[] args){
     byte age =21;
     System.out.println(age);
}
}


Output:-

21

Range

byte  --- [-128,127]

short --- [-32k,32k]

int   --- [-2Billion,2Billion]

long  --- [above 2 billion]

________
3.long
_________

Program:-
_________

Input:-

public class Datatypes{
  public static void main(String[] args){
     long age =21000000000000000000l;
     System.out.println(age);
}
}


Output:-

210000000000000000000l

Note:- We should add l at last of the given long number otherwise it will consider the value as int.
